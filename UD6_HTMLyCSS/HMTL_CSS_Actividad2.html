Actividad: Estructurar y estilizar una página web con secciones y tablas

Objetivo

Crear una página web estructurada con divisiones claras utilizando etiquetas HTML5, tablas básicas, y atributos como id y class. Se aplicarán estilos con CSS para organizar y personalizar la presentación.

Parte 1: Configuración inicial

Paso 1: Preparar el entorno

  1. Crea una carpeta llamada actividad2.
  2. Dentro de esta carpeta, crea los siguientes archivos:
    • estructura.html
    • estilos.css

Parte 2: Diseñar la estructura HTML

Paso 2: Crear la página principal

Escribe el siguiente código en estructura.html:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Estructura Web</title>
    <link rel="stylesheet" href="estilos.css">
</head>
<body>
    <!-- Cabecera de la página -->
    <header id="cabecera">
        <h1>Mi página web estructurada</h1>
        <nav>
            <ul>
                <li><a href="#seccion1">Inicio</a></li>
                <li><a href="#seccion2">Contenido</a></li>
                <li><a href="#footer">Contacto</a></li>
            </ul>
        </nav>
    </header>

    <!-- Contenido principal -->
    <main>
        <!-- Sección 1: Introducción -->
        <section id="seccion1" class="seccion">
            <h2>Bienvenida</h2>
            <p>Este es el inicio de mi página web estructurada con HTML5 y CSS.</p>
        </section>

        <!-- Sección 2: Contenido con tablas -->
        <section id="seccion2" class="seccion">
            <h2>Datos importantes</h2>
            <p>Esta sección incluye una tabla básica para organizar la información:</p>
            <table>
                <thead>
                    <tr>
                        <th>Nombre</th>
                        <th>Edad</th>
                        <th>Ocupación</th>
                    </tr>
                </thead>
                <tbody>
                    <tr>
                        <td>Ana</td>
                        <td>25</td>
                        <td>Desarrolladora</td>
                    </tr>
                    <tr>
                        <td>Carlos</td>
                        <td>30</td>
                        <td>Diseñador</td>
                    </tr>
                    <tr>
                        <td>Luisa</td>
                        <td>28</td>
                        <td>Gestora de proyectos</td>
                    </tr>
                </tbody>
            </table>
        </section>
    </main>

    <!-- Pie de página -->
    <footer id="footer">
        <p>Contacto: ejemplo@correo.com</p>
    </footer>
</body>
</html>

Parte 3: Estilizar con CSS

Paso 3: Añadir estilos en estilos.css

Escribe el siguiente código en estilos.css:

/* Estilo general */
body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
    background-color: #f4f4f4;
    color: #333;
}

/* Encabezado */
header {
    background-color: #4CAF50;
    color: white;
    padding: 20px;
    text-align: center;
}

header nav ul {
    list-style: none;
    padding: 0;
}

header nav ul li {
    display: inline;
    margin: 0 10px;
}

header nav ul li a {
    color: white;
    text-decoration: none;
}

/* Secciones */
.seccion {
    padding: 20px;
    margin: 20px auto;
    background-color: #fff;
    border: 1px solid #ddd;
    border-radius: 5px;
    max-width: 800px;
}

/* Tablas */
table {
    width: 100%;
    border-collapse: collapse;
    margin: 20px 0;
}

table th, table td {
    border: 1px solid #ddd;
    padding: 8px;
    text-align: center;
}

table th {
    background-color: #4CAF50;
    color: white;
}

/* Pie de página */
footer {
    background-color: #333;
    color: white;
    text-align: center;
    padding: 10px 0;
    position: relative;
}

Parte 4: Actividades prácticas

Actividad 1: Personaliza tu página

  • Cambia los colores de fondo del encabezado y del pie de página en el archivo CSS.
  • Añade una fila adicional a la tabla con datos ficticios.
  • Añade una columna adicional a la tabla con datos ficticios.
  • Añade una nueva sección que incluya una imagen y texto explicativo.

Actividad 2: Reflexión

Responde las siguientes preguntas en un documento llamado reflexion_estructura.txt:

  • ¿Qué aprendiste al trabajar con secciones y tablas?
  • ¿Qué te pareció más difícil de implementar y por qué?
